# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful, harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community

Unacceptable behavior includes:

- Harassment, abuse, or personal attacks
- Trolling, insulting/derogatory comments, and political flamewars
- Public or private harassment
- Publishing othersâ€™ private information without consent

## Responsibilities

Project maintainers are responsible for clarifying standards and taking appropriate, fair action in response to unacceptable behavior.

## Scope

This Code applies within project spaces and when an individual is representing the project in public spaces.

## Enforcement

Report abusive, harassing, or otherwise unacceptable behavior:

- Prefer GitHub Security Advisories or a private email to the maintainers
- If in doubt, open an issue asking for a private contact channel (without details)

Maintainers will review and respond as appropriate, up to and including temporary or permanent bans.

## Attribution

This Code of Conduct is inspired by community standards such as the Contributor Covenant, adapted for this project.
